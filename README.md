# AI Media - 智能媒体处理平台

基于 Flask 和 Cloudinary 构建的云端媒体管理平台，提供图片上传、智能抠图、图片缩放等功能，拥有简洁直观的中文界面。

## 功能特性

- **云端媒体存储** — 图片和视频上传至 Cloudinary 云端存储
- **智能抠图** — 一键去除图片背景
- **图片缩放** — 自定义尺寸智能缩放
- **实时画廊** — 上传的媒体实时动态展示
- **响应式设计** — 适配各种设备的现代界面

## 技术栈

- Python 3.x / Flask
- Cloudinary SDK（云端媒体管理）
- HTML / CSS / JavaScript

## 快速开始

```bash
git clone https://github.com/woogowoogo/aimedia-platform.git
cd aimedia-platform
pip install -r requirements.txt
python main.py
```

浏览器访问 `http://127.0.0.1:5000/`
